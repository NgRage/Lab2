# Задание 1

## Задача 1

### Текст задачи

Получить список из минимальных цифр натуральных чисел, содержащихся в
исходном списке


### Алгоритм решения

1. Запросить у пользователя натуральное число N для указания количества элементов списка + проверка
2. Сформировать список из N элементов введенных пользователем + проверка
3. При помощи list.map пройтись по каждому элементу и найти в каждом минимальную цифру
4. Сформирвоать новый список из минимальных цифр
5. Результат вывести на экран
